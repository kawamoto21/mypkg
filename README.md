#　mypkg

##　countupコマンド
![test](https://github.com/kawamoto21/mypkg/actions/workflows/test.yml/badge.svg)

◦使い方
2端末使用します。
①　片方の端末で$ ros2 run mypkg talker
②　もう一方で　$ ros2 run mypkg listener
*終了するときは　Ctrl + C
