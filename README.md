# mypkg
ロボットシステム学の課題2で作成したものです。

## countupコマンド
!test

数字をカウント＋現在時刻を表示

◦使い方
2端末使用します。
1. 片方の端末で `$ ros2 run mypkg talker`
2. もう一方で `$ ros2 run mypkg listener`
*終了するときは `Ctrl + C`

- Python
